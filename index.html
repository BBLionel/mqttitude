Longitude & co. Decentralized.

MQTTitude allows a device to periodically publish its location status to an MQTT broker. If you've previously used Latitude, you can think of this as a a decentralized Latitude. MQTTitude was started because Latitude was de-comissioned in August 2013.

MQTT. Huh?

For the "Internet of Things".

MQTT is a machine-to-machine connectivity protocol which was designed as a very lightweight publish/subscribe messaging transport. It is very useful for connections with remote locations (such as from a mobile device) where network bandwidth is at a premium. MQTTitude uses the MQTT protocol to safely transmit your location information to your server (which is called a "broker").
